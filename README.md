# SocialBox
SocialBox is a Bruteforce Attack Framework [ Facebook , Gmail , Instagram ,Twitter ] , Coded By Belahsan Ouerghi
# Installation
```
apt-get update
apt-get install git
git clone https://github.com/ibrahim21/SocialBox.git 
cd SocialBox
chmod +x install-sb.sh
./install-sb.sh
```
# Run
```
./SocialBox.sh
```
